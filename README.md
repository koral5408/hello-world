# 2、关于列表
## 无序列表，使用星号、加号和减号作为列表标记（加减乘）

### 星号*
* 红
* 绿
* 蓝

### 加号+
+ 红
+ 绿
+ 蓝

### 减号-
- 红
- 绿
- 蓝
